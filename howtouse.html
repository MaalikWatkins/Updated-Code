How to Use

This website is intended to help monitor the vital signs of people in need of having this information tracked. There are a variety of tabs on Welcome Page, which are: “Home,” “How to Use,” “Contact,” and “Login.” To get started the user presses the Login tab which takes the user to the sign-up screen. If this is the first use of the website, the user inputs their email address and creates a password which will be used for subsequent logins. The user then presses the sign-up tab highlighted in red for storing the email address and password. On all subsequent sign-ins, the user inputs their email address and password and presses the Login tab, which is also highlighted in red.
